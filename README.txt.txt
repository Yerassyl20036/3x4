INSTALLATION AND RUNNING INSTRUCTIONS

1. Ensure Python is installed on your system. You can download it from https://www.python.org/downloads/.

2. Open a command line interface and navigate to the project directory.

3. Create a Python virtual environment:
   Windows:
   python -m venv venv
   .\venv\Scripts\activate

   macOS/Linux:
   python3 -m venv venv
   source venv/bin/activate

4. Install the required packages:
   pip install -r requirements.txt

5. Run the script:
   python 3x4.py

Note: Replace '3x4.py' with the actual name of your Python script.
